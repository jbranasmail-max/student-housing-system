# 🏠 Student Housing Management System

## نظام إدارة السكن الطلابي

نظام إدارة السكن الطلابي هو نظام ويب متكامل مبني باستخدام **Django Framework**، يهدف إلى تنظيم وإدارة السكن الطلابي بطريقة إلكترونية حديثة.

يساعد النظام إدارة السكن على متابعة الطلاب، الغرف، الحجوزات، المدفوعات، الموظفين، وطلبات الدعم من خلال لوحة تحكم سهلة الاستخدام.

---

# ✨ مميزات النظام

✅ إدارة حسابات المستخدمين والصلاحيات.  
✅ إدارة بيانات الطلاب.  
✅ إدارة المباني والغرف السكنية.  
✅ تسجيل وتخصيص حجوزات الطلاب.  
✅ متابعة إشغال الغرف.  
✅ إدارة المدفوعات والفواتير.  
✅ إدارة الموظفين.  
✅ لوحة تحكم للإحصائيات والمتابعة.  
✅ نظام الدعم والاستفسارات.  
✅ واجهة استخدام سهلة ومنظمة.
